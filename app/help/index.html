Sailwatch is a simple tool for executing starts using rule 28 with 5-4-1-0 intervals and taking finish times.
  • Start
  • Finish
  • Note

Start

The icon opens a dialog for registering a start. A few Suggestions for start times are shown, starting 330 seconds after now or the latest registered event. Similarly, fleets to start can be selected. New fleets can be added using the input field.

Finish

The icon always hovers at the lower left corner.
